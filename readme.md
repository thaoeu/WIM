# 特性

1. 基于 LTSC2019 封装；

1. 关闭了 Windows 更新；

1. 关闭了时间线（频繁读取硬盘）；

1. 默认启用最佳性能（可在此电脑、属性、高级系统设置、性能中关闭）；

1. 为防止安装后系统重置默认应用频繁弹窗，移除音视频软件，请用户根据习惯自行安装；

## 包含软件：

-	QuickLook	作用：选中文件后按下空格键预览，包括且不限于图片、视频及文档**；**

-	火绒安全软件及其弹窗插件；

-	Bandzip 6.29（最后一个无广告版本）


注意：此版本不含 Windows 应用商店，默认无法使用 UWP 应用，如有使用新版 Edge、wsl 等应用需求谨慎使用；

默认用户密码为 0998；

UAC 调整为从不通知，如需通知请自行开启。
